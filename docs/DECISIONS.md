# Decisions

## Frozen Phase 1 Stack

- Frontend: Next.js, TypeScript, Tailwind CSS, shadcn/ui, TanStack Query.
- Backend: Node.js, Express, TypeScript.
- Database: MongoDB Atlas.
- Queue: BullMQ with Upstash Redis.
- AI: Groq behind a provider abstraction.
- Search: Tavily and Serper behind a unified abstraction.
- Storage: Cloudinary.
- PDF: local XeLaTeX.
- Logging: Pino.

Any deviation from this stack should be recorded here before implementation.
