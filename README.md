# AI Resume Tailor and Prep

AI-powered career preparation platform for resume management, ATS tailoring, interview research, practice workspaces, learning analytics, and mobile quick revision.

## Current Milestone

Milestone 1 starts with the environment skeleton and authentication foundation.

## Planned Commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm test
```

Copy `.env.example` to `.env` and fill in local development values before running the API.
